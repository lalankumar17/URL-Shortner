# URL Shortner

ShortURL is a free tool to shorten URLs, create easy-to-share short links, and generate QR codes for a given URL.

A URL shortener is a useful tool that converts long and complicated URLs into short, clean links that are easier to share and use.

## About This Website

- Shorten long URLs using TinyURL
- Create custom short links
- Generate QR codes for a given URL
- Copy short links easily

## How to Run Locally

1. Install Node.js.
2. Open terminal in this project folder.
3. Run:
```bash
cd node-backend
```

```bash
npm install
```

```bash
npm start
```

Then open `http://localhost:3000` in your browser.

Optional: add `TINYURL_API_TOKEN=your_tinyurl_api_token` in a root `.env` file.
